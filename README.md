# Aenzbi Smart Contract Library

An advanced, hyper-optimized smart contract library bridging enterprise security with cutting-edge EIPs. Designed to be a next-generation alternative to OpenZeppelin.

## Core Features
* **Gas Optimized:** Custom errors and unchecked math blocks.
* **Transient Storage (EIP-1153):** Cheaper reentrancy guards utilizing `TSTORE`.
* **Native EIP-2612:** Gasless permit approvals built-in.
